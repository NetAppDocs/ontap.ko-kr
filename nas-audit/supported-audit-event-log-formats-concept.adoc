= ONTAP 감사 이벤트 로그에 지원되는 형식에 대해 알아봅니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
변환된 감사 이벤트 로그에 지원되는 파일 형식은 evtx 및 XML 파일 형식입니다.

감사 구성을 만들 때 파일 형식의 유형을 지정할 수 있습니다. 기본적으로 ONTAP는 바이너리 로그를 'evtx' 파일 형식으로 변환합니다.
