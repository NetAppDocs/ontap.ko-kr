= 액세스 검사 유형 보안 추적 모니터
:icons: font
:imagesdir: ../media/


[role="lead"]
파일 또는 폴더에 대한 액세스 검사는 여러 기준에 따라 수행됩니다. Security Trace는 이러한 모든 기준에 대한 작업을 모니터링합니다.

보안 추적 모니터가 사용하는 액세스 검사의 유형은 다음과 같습니다.

* 볼륨 및 qtree 보안 스타일
* 작업이 요청된 파일 및 폴더가 포함된 파일 시스템의 효과적인 보안
* 사용자 매핑
* 공유 수준 권한
* 내보내기 수준 권한
* 파일 수준 권한
* 스토리지 레벨 액세스 가드 보안
