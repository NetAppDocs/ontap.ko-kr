[role="lead"]
ONTAP 9.8부터 네트워크에 대해 System Manager에 표시되는 데이터를 다운로드할 수 있습니다.

List View _ 에 정보를 표시하면 * Download * 를 클릭하면 표시되는 객체 목록이 다운로드됩니다.

* 목록은 CSV(쉼표로 구분된 값) 형식으로 다운로드됩니다.
* 표시된 열의 데이터만 다운로드됩니다.
* CSV 파일 이름의 형식은 개체 이름과 타임 스탬프로 지정됩니다.
