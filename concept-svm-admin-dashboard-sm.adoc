= ONTAP System Manager SVM 관리자 대시보드에 대해 알아보십시오
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.19.1부터 SVM 관리자는 전용 System Manager 대시보드와 Storage > Volumes 페이지를 사용하여 SVM을 모니터링하고 관리할 수 있습니다. 이는 테넌트 ONTAP 환경에서 클러스터 관리자가 각 테넌트에 대한 SVM 대시보드를 제공할 수 있는 경우 특히 유용합니다.

클러스터 관리자는 SVM 관리자에게 로그인 자격 증명을 제공합니다. SVM 관리자는 SVM 대시보드와 *스토리지 > 볼륨* 페이지에서 세부 정보와 성능을 확인할 수 있습니다. 대시보드에는 SVM 프로토콜, 네트워크 인터페이스, 세부 정보 및 성능 지표가 표시됩니다. 볼륨 페이지에는 SVM 볼륨에 대한 정보가 제공되며, 클러스터 관리자가 허용하는 경우 SVM 관리자가 새 볼륨을 추가할 수 있습니다.
