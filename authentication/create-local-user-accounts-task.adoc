= 로컬 계정 액세스 개요를 활성화합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
로컬 계정은 계정 정보, 공개 키 또는 보안 인증서가 스토리지 시스템에 상주하는 계정입니다. 'Security login create' 명령을 사용하여 로컬 계정에서 admin 또는 data SVM에 액세스할 수 있습니다.
