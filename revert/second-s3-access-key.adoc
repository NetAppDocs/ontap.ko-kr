= ONTAP 9.19.1에서 되돌리기 전에 두 번째 S3 액세스 키를 제거하십시오
:allow-uri-read: 


[role="lead"]
ONTAP 9.19.1로 업그레이드한 후 사용자에게 두 번째 S3 액세스 키를 추가한 경우, 이전 ONTAP 릴리스로 되돌리려면 먼저 이러한 키를 제거해야 합니다.

.단계
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["S3 액세스 키 표시"] S3 사용자에게 두 번째 액세스 키가 추가되었는지 여부를 확인합니다.
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["두 번째 액세스 키 제거"] 모든 S3 사용자용.
