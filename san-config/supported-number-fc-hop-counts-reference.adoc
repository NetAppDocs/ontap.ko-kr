= 지원되는 FC 홉 수
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
호스트와 스토리지 시스템 간에 지원되는 최대 FC 홉 수는 스위치 공급업체와 FC 구성에 대한 스토리지 시스템 지원에 따라 다릅니다.

홉 수는 이니시에이터(호스트)와 타겟(스토리지 시스템) 사이의 경로에 있는 스위치의 수로 정의됩니다. Cisco는 이 값을 SAN fabric_의 _DIAMETER로 지칭하기도 합니다.
