= System Manager를 사용하여 ONTAP SVM 관리 및 모니터링
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.19.1부터 SVM 관리자는 전용 System Manager 대시보드를 사용하여 자체 SVM을 모니터링하고 관리할 수 있습니다.

.시작하기 전에
클러스터 관리자가 다음 정보를 제공해야 합니다.

* SVM에 대한 관리 IP 주소
* SVM System Manager 대시보드의 자격 증명(사용자 이름 및 암호)(vsadmin 사용자는 잠금 해제되어 있어야 합니다)
* 선택적으로 SVM에 볼륨을 추가할 수 있는 권한


.단계
. 브라우저에 SVM의 관리 IP 주소를 입력하십시오.
. SVM 관리자(vsadmin)의 사용자 이름과 암호를 입력하십시오.
. *Sign in*을 선택합니다.
+
ONTAP System Manager 대시보드가 열립니다.

. SVM에 대한 다음 정보를 확인할 수 있습니다.
+
[cols="1,2"]
|===
| SVM 정보 | 설명 


| 프로토콜 | SVM에서 활성화된 프로토콜입니다. 


| 세부 | NIS 및 LDAP 정보, 언어, 그리고 볼륨 삭제 후 보존 기간. 


| 성능 | 선택한 프로토콜에 대한 시간별, 일별, 주별, 월별 또는 연도별 지연 시간, IOPS 및 처리량 성능 통계. 
|===
. 필요에 따라 데이터를 쉼표로 구분된 값(csv) 파일로 다운로드하여 추가 분석을 진행할 수 있습니다.
. 스토리지 > 볼륨 * 을 선택합니다.
+
볼륨 대시보드 페이지가 열립니다.

+
클러스터 관리자가 SVM에 볼륨을 추가할 수 있는 권한을 부여할 수 있습니다. 이 권한이 부여되지 않으면 *추가* 버튼을 선택할 수 없으며 오류 메시지가 표시됩니다.
