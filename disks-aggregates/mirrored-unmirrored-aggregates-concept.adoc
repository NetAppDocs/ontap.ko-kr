= 미러링되지 않은 애그리게이트
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
_SyncMirror_라는 옵션 기능을 사용하여 서로 다른 RAID 그룹에 저장된 복사본 또는 _Plex_의 애그리게이트 데이터를 동기식으로 미러링할 수 있습니다. 플렉스는 RAID 유형보다 더 많은 디스크에 장애가 발생하거나 RAID 그룹 디스크에 대한 연결이 끊어지면 데이터 손실을 방지합니다.
