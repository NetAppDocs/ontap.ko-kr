[role="lead"]
'스토리지 애그리게이트' 및 '볼륨' 명령을 사용하여 애그리게이트 및 볼륨, 스냅샷 복사본에 사용 중인 공간이 어떤 것인지 확인할 수 있습니다.

|===


| 정보를 표시하려면... | 이 명령 사용...

Aggregate는 사용 가능한 공간 비율, 스냅샷 예약 크기 및 기타 공간 사용 정보에 대한 세부 정보를 포함합니다
 a|

 a|

 a|

 a|
