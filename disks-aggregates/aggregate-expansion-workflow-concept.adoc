= ONTAP에서 로컬 계층에 용량을 추가하는 워크플로
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
로컬 계층에 용량을 추가하려면 먼저 추가할 로컬 계층을 식별하고, 필요한 새 스토리지를 결정하고, 새 디스크를 설치하고, 디스크 소유권을 할당하고, 필요한 경우 새 RAID 그룹을 생성해야 합니다.

System Manager 또는 ONTAP CLI를 사용하여 용량을 추가할 수 있습니다.
