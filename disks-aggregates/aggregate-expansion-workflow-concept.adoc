= 로컬 계층에 용량을 추가하는 워크플로우(애그리게이트 확장)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
용량을 로컬 계층에 추가(Aggregate 확장)하려면 먼저 추가할 로컬 계층을 식별하고, 새 스토리지가 필요한 용량을 확인하고, 새 디스크를 설치하고, 디스크 소유권을 할당하고, 필요한 경우 새 RAID 그룹을 생성해야 합니다.

System Manager 또는 CLI를 사용하여 용량을 추가할 수 있습니다.
