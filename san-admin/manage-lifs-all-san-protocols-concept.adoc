= SAN 환경을 위한 ONTAP LIF에 대해 알아보십시오
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 시스템의 LUN과 네임스페이스는 스토리지 가상 머신(SVM) 내에 저장됩니다. 네트워크 인터페이스(LIF)를 통해 SAN 클라이언트는 SVM에 연결하여 SVM에 포함된 LUN 및 네임스페이스의 데이터에 액세스할 수 있습니다.

* SAN을 지원하는 모든 스토리지 가상 머신(SVM)에 대해 하나의 관리 LIF를 구성해야 합니다.
* 노드당 두 개의 LIF를 구성할 수 있으며, 하나는 FC에 사용되는 각 패브릭용이고 다른 하나는 iSCSI용 이더넷 네트워크 분리용입니다.
* 연결 방식으로는 직접 연결 또는 이더넷 스위치를 사용하는 방식이 지원됩니다. 두 가지 연결 방식 모두에 대해 LIF를 구성해야 합니다.
