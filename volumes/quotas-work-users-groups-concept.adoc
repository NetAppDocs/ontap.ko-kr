= 사용자 및 그룹에서 할당량이 작동하는 방식에 대한 개요입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용자 또는 그룹을 할당량의 타겟으로 지정할 수 있습니다. 할당량을 정의할 때 고려해야 할 몇 가지 구축 차이점이 있습니다.

알아야 할 몇 가지 차이점이 있습니다.

* 사용자 또는 그룹
* Unix 또는 Windows입니다
* 특수 사용자 및 그룹
* 여러 ID가 포함되어 있습니다


환경에 따라 사용자의 ID를 지정하는 방법도 다양합니다.
