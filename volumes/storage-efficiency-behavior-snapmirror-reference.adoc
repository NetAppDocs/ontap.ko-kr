= 볼륨 이동 및 SnapMirror 작업에 대한 스토리지 효율성 동작
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
볼륨 이동 또는 SnapMirror 작업을 수행할 때 볼륨에서 스토리지 효율성이 동작하는 방식 그리고 SnapMirror 휴식을 수행하고 온도에 민감한 스토리지 효율성을 수동으로 활성화할 때 발생하는 상황은 소스 볼륨의 효율성 유형에 따라 달라집니다.
