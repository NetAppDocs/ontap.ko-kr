= Qtree에서 할당량 작동 방식
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
qtree를 타겟으로 사용하여 할당량을 생성할 수 있습니다. 이러한 할당량을 _tree quotas_라고 합니다. 특정 qtree에 대해 사용자 및 그룹 할당량을 생성할 수도 있습니다. 또한 FlexVol 볼륨의 할당량은 해당 볼륨에 포함된 qtree에 의해 상속되는 경우도 있습니다.
