= 디렉토리 삭제 작업을 취소합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
디렉토리 삭제를 취소해야 하는 경우 작업이 진행 중인 동안 취소할 수 있습니다.

.단계
. 고급 권한 모드 시작:
+
프리빌리지 어드밴스드

. 디렉토리 삭제가 진행 중인지 확인합니다.
+
'볼륨 파일 비동기 삭제 표시'

+
디렉토리의 SVM, 볼륨, jobid 및 경로가 표시되면 작업을 취소할 수 있습니다.

. 디렉토리 삭제 취소:
+
'볼륨 파일 async-delete cancel-vserver SVM_name-volume volume_name-jobid job_id'
