= 볼륨 효율성 모드를 확인합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
AFF 볼륨에서 'volume-Efficiency-show' 명령을 사용하여 효율성이 설정되었는지 확인하고 현재 효율성 모드를 확인할 수 있습니다.

.단계
. 볼륨의 효율성 모드를 확인합니다.
+
'* 볼륨 효율성 표시 - vserver <vserver name> - volume <volume name> - fields storage-Efficiency-mode * '
