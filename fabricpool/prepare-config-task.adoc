= FabricPool 구성 개요를 준비합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
FabricPool를 구성하면 데이터의 액세스 빈도 여부를 기준으로 저장할 스토리지 계층(로컬 성능 계층 또는 클라우드 계층) 데이터를 관리할 수 있습니다.

FabricPool 구성에 필요한 준비는 클라우드 계층으로 사용하는 오브젝트 저장소에 따라 다릅니다.
