= 스캐너 풀 개요를 구성합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
스캐너 풀은 SVM에 연결할 수 있는 Vscan 서버 및 특별 권한 사용자를 정의합니다. 스캐너 정책은 스캐너 풀이 활성 상태인지 여부를 결정합니다.

[NOTE]
====
SMB 서버에서 내보내기 정책을 사용하는 경우 각 Vscan 서버를 엑스포트 정책에 추가해야 합니다.

====
