= 원격 사무소에서 BranchCache 구성에 대한 정보를 찾을 수 있는 위치
:icons: font
:imagesdir: ../media/


[role="lead"]
CIFS 서버에서 BranchCache를 구성한 후에는 클라이언트 컴퓨터와 원격 사무실의 캐싱 서버에 BranchCache를 설치하고 구성해야 합니다(선택 사항). Microsoft는 원격 사무소에서 BranchCache를 구성하는 지침을 제공합니다.
