= SnapMirror 관계에 대한 호환 ONTAP 버전
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
소스 및 타겟 볼륨에서 SnapMirror 데이터 보호 관계를 생성하기 전에 호환되는 ONTAP 버전을 실행해야 합니다. ONTAP를 업그레이드하기 전에 현재 ONTAP 버전이 SnapMirror 관계에 대한 대상 ONTAP 버전과 호환되는지 확인해야 합니다.

[NOTE]
====
SVM 복제에서는 버전에 상관없이 지원되지 않습니다.

====

== SnapMirror SVM 재해 복구 관계

SVM 재해 복구 데이터 및 SVM 보호:

SVM 재해 복구는 동일한 버전의 ONTAP를 실행하는 클러스터 간에만 지원됩니다.

SVM 마이그레이션을 위한 SVM 재해 복구:

* 복제는 원본의 이전 버전의 ONTAP에서 대상의 동일한 버전 이상의 ONTAP 버전(예: ONTAP 9.11.1에서 ONTAP 9.12.1로)으로 한 방향으로 지원됩니다.
* 타겟 클러스터의 ONTAP 버전은 아래 표와 같이 두 개 이상의 최신 버전이 아니어야 합니다.
* 장기 데이터 보호 사용 사례에는 복제가 지원되지 않습니다.


릴리스 버전 뒤의 별표(*)는 클라우드 전용 릴리스를 나타냅니다.

지원을 확인하려면 왼쪽 테이블 열에서 원본 버전을 찾은 다음 맨 위 행에서 대상 버전을 찾습니다.
