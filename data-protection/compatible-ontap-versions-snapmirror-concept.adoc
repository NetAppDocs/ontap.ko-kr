= SnapMirror 관계에 대한 호환 ONTAP 버전
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]

[NOTE]
====
SVM 복제에서는 버전에 상관없이 지원되지 않습니다.

====
