= CLI 세션의 레코드를 관리하는 명령입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
'system script' 명령어를 이용하여 CLI 세션 기록을 관리한다.

|===
| 원하는 작업 | 이 명령 사용... 


 a| 
지정된 파일에 현재 CLI 세션 녹화를 시작합니다
 a| 
'시스템 스크립트 시작

현재 CLI 세션 녹화를 중지합니다
 a| 
'시스템 스크립트 중지'

'시스템 스크립트 쇼'



 a| 
CLI 세션의 레코드를 FTP 또는 HTTP 대상으로 업로드합니다
 a| 
'시스템 스크립트 업로드'
