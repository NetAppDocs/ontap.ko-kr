* * 메시지 *
+
"전송 실패(CFO가 중단을 시작함)"

* * 원인 *
+
SMTape 작업은 CFO 애그리게이트의 스토리지 페일오버(테이크오버 및 반환) 작업 때문에 중단됩니다.

* * 시정 조치 *
+
CFO 애그리게이트의 스토리지 페일오버가 완료된 후 SMTape 작업을 수행합니다.
