= 스냅샷을 찾지 못했습니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 메시지 *
+
스냅샷을 찾지 못했습니다

* * 원인 *
+
백업에 지정된 스냅샷 복사본을 사용할 수 없습니다.

* * 시정 조치 *
+
지정된 스냅샷 복사본을 사용할 수 있는지 확인합니다. 그렇지 않은 경우 올바른 스냅샷 복사본으로 재시도하십시오.
