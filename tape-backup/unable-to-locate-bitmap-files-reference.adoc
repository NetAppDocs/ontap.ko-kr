= 비트맵 파일을 찾을 수 없습니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 메시지 *
+
비트맵 파일을 찾을 수 없습니다

* * 원인 *
+
