= 입력 테이프가 잘못되었습니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 메시지 *
+
잘못된 입력 테이프

* * 원인 *
+
백업 이미지의 서명이 테이프 헤더에서 유효하지 않습니다. 테이프에 손상된 데이터가 있거나 유효한 백업 이미지가 없습니다.

* * 시정 조치 *
+
유효한 백업 이미지로 복원 작업을 다시 시도하십시오.
