= file name> 파일을 복원하지 못했습니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 메시지 *
+
파일 이름을 복원하지 못했습니다

* * 원인 *
+
대상 볼륨에 있는 LUN의 파일 이름과 동일한 파일 이름의 DAR(Direct Access Recovery)이 수행된 경우 DAR가 실패합니다.

* * 시정 조치 *
+
파일의 DAR를 다시 시도하십시오.
