= 진행 중인 최대 허용 덤프 또는 복원 수(최대 세션 제한)
:icons: font
:imagesdir: ../media/


* * 메시지 *
+
허용되는 덤프 또는 복원 횟수 중 최대값 _ (최대 세션 제한) _ 이(가) 진행 중입니다

* * 원인 *
+
최대 백업 또는 복원 작업 수가 이미 실행 중입니다.

* * 시정 조치 *
+
현재 실행 중인 일부 작업이 완료된 후 작업을 재시도하십시오.
