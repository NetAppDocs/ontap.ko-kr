= 하드 마운트 사용
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
장착 문제를 해결할 때 올바른 장착 유형을 사용하고 있는지 확인해야 합니다. NFS는 소프트 마운트와 하드 마운트의 두 가지 마운트 유형을 지원합니다. 안정성을 위해 하드 마운트만 사용해야 합니다.

소프트 마운트를 사용하면 안 됩니다. 특히, NFS 시간 제한이 자주 발생할 가능성이 있습니다. 이러한 시간 초과로 인해 경합 상태가 발생하여 데이터가 손상될 수 있습니다.
