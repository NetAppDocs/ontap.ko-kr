= ONTAP에서 NFSv4 ACL을 사용하여 파일을 삭제할 수 있는지 여부를 확인하는 방법
:icons: font
:imagesdir: ../media/


[role="lead"]
파일을 삭제할 수 있는지 여부를 확인하기 위해 ONTAP에서는 파일의 삭제 비트와 포함하는 디렉토리의 delete_child 비트를 함께 사용합니다. 자세한 내용은 NFS 4.1 RFC 5661을 참조하십시오.
