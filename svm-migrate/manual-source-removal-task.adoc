= 클라이언트 전환 후 소스 ONTAP SVM을 수동으로 제거합니다.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
소스 정리를 사용하지 않도록 설정한 상태에서 SVM 마이그레이션을 수행한 경우, 클라이언트 컷오버가 완료된 후 소스 SVM을 수동으로 제거할 수 있습니다.

.단계
. 소스 정리를 위해 상태가 준비되었는지 확인하세요.
+
[source, cli]
----
vserver migrate show
----
. 소스 정리:
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----
