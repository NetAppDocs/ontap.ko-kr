= ONTAP 볼륨 마이그레이션 모니터링
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
'vserver migrate show' 명령을 사용하여 전체 SVM 마이그레이션을 모니터링하는 것 외에도 SVM에 포함된 볼륨의 마이그레이션 상태를 모니터링할 수 있습니다.

.단계
. 대상 클러스터에서 볼륨 마이그레이션 상태를 확인합니다.
+
[source, cli]
----
vserver migrate show-volume
----
